Zomato Data Analysis (EDA Project)

This project performs Exploratory Data Analysis (EDA) on the Zomato dataset to uncover insights about restaurant trends, customer ratings, geographical distribution, and service availability across countries and cities.

📁 Project Overview

The objective of this project is to analyze restaurant data and identify key business insights that can help understand:

Customer rating behavior

Popular cuisines and cities

Distribution of restaurants across countries

Online delivery availability

Pricing patterns and cost trends

The dataset contains ~9551 records and 21+ features including restaurant details, location, ratings, and services.

🛠️ Tools & Technologies Used

Python

Pandas & NumPy

Matplotlib & Seaborn

Jupyter Notebook

🔍 Key Steps Performed
1. Data Cleaning & Preprocessing

Checked for missing values (none found)

Handled encoding issues (latin1)

Merged country dataset with main dataset

2. Data Exploration

Statistical summary of numerical variables

Distribution of cities and countries

Feature analysis (cost, ratings, votes)

3. Data Visualization

Pie charts (country-wise distribution)

Bar plots (rating distribution)

Box plots (rating spread)

Count plots (rating colors)

📈 Key Insights & Observations
🌍 Country-wise Analysis

Majority of transactions (~94%) are from India

Followed by USA and UK

⭐ Ratings Analysis

Most ratings fall between 2.5 – 3.9 (Average to Good)

Very few restaurants have extreme low or very high ratings

Large number of restaurants are not rated, indicating user engagement gap

📦 Online Delivery Insights

Online delivery is available only in India and UAE

India dominates in both usage and service availability

🏙️ City-wise Insights

New Delhi has the highest number of restaurants (~68%)

Followed by Gurgaon, Noida, and Faridabad

📊 Rating Distribution

Median rating is approximately 3.3

Majority of restaurants fall under:

Average (2.5–3.4)

Good (3.5–3.9)

📌 Business Implications

Zomato has strong market dominance in India but limited international penetration

High number of unrated restaurants indicates opportunity for engagement strategies

Mid-range ratings suggest scope for improving service quality

Online delivery expansion can be a major growth driver outside India

🚀 Conclusion

This analysis highlights that Zomato's ecosystem is heavily concentrated in India, with moderate customer satisfaction levels and significant potential for growth in engagement, service quality, and international markets.
